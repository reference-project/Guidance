入院须知

为了给患者创造安静、安全、舒适、无烟健康，有利于疾病恢复的休养环境，减轻家属负担；预防院内感染的发生，我院在提供优质服务的基础上实施无陪伴制度。请您协助做好以下各项工作：

1.为了保证患者安全，请告知患者不要随意下床活动，如有需要可使用呼叫器呼叫护士／护工，护士／护工也会按时巡视。

2.每日探视时间为16:00一19:30，为保护病情隐私权，医生每天16:00至17:30 向患者或患者家属介绍患者病情。

3.患者住院期间，如突然出现病情变化，医务人员将通知家属，请立即到院。

4.医务人员会在探视时间交待费用情况，请及时补充住院费用，以免影响治疗。

5.住院患者不得自行离开病区、医院，擅自外出者，由此产生的一切后果自负。

6.禁止携带锐器，不得私自使用各种电器，否则后果自负。

7.患者应妥善保管好自己的财物，丢失自负。并请自觉爱护医院设施，损坏照价赔偿。

8.为了减少院内感染的发生，请不要随意到其他病室或床单位处逗留；卧床期间护士／护工会给您拉起床档，请不要自行调节，如有需要可呼叫护士／护工。

9.您已进入无烟医院，在住院期间，病人及探视家属禁止在病区内吸烟，禁止饮酒及大声喧哗。

10.入院后在处置室预交生活护理费10天（）元，出院时多退少补

医疗膳食配置室简介

天津医院医疗膳食配制室是天津市第一家管办分离的医疗膳食配制室，实行无陪护包餐制，是天津医院无陪护服务的重要组成部分。医疗膳食配制室与医院营养科密切协作，为就医患者住院期间提供科学合理的膳食服务。

医疗膳食配制室现有工作人员 80 余人，含专业管理人员、营养技师、营养烹调师、面点师、切配工、配膳员、洗消工等，负责全院日均 1000 多患者的膳食配制工作。

膳食种类：普通营养膳食、软质膳食、流质膳食、半流质膳食、回民营养膳食、特殊需求膳食（小炒、包子、饺子等）、治疗膳食中的糖尿病膳食、低盐低脂膳食、肠内营养素配置以及匀浆膳。同时,我们努力推进膳食科别化，目前已经针对儿科开展儿童膳食、妇产科开展月子餐、国际诊疗科开展VIP餐。

膳食配制室以平衡膳食为原则，食谱的制定加工发放实行规范量化管理。营养技师协同院营养科营养师负责各类膳食食谱制定和全程质量监督，指导营养烹调师按要求制作，保证营养食谱质量要求和食品卫生的有效落实。其中，治疗膳食由院营养科营养师遵照临床医嘱，根据患者自身情况，进行个体化营养食谱的设计，营养膳食烹调师严格按照营养科下发的医嘱单执行、制作。

充分发挥营养膳食治疗第二药房作用。

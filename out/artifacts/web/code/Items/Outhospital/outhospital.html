  • 复印病例及递送代办

    复印病例，一般情况要等到患者出院一段时间之后自行来医院办理，手续繁琐，因此,我公司开通复印病例及递送的服务，您只需提供相关信息及材料交给我们的客服专员即可，解决了患者或家属奔波。具体条件详见须知。

    8:00-17:00
    022-60858531
    须知
    1

    预约办理流程：

    ①携带患者本人身份证原件（儿科患者需本人户口本原件或病案内有签字的家属身份证原件）自行到病案科预约登记并取得凭证。
